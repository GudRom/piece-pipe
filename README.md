# Piece Pipe

This is the concept of an application that allows you to collect a shared playlist with your loved ones and add it to your music service.

Technology Stack
---

- ReactJS
- Redux Toolkit
- React-Router
- Typescript
- SCSS
- json-server
- Vite

Local run
---

1. Install dependencies:

    ```bash
    npm i
    ```
2. Start app: 
    ```bash
    npm start
    ```
