# Piece Pipe

This is the concept of an application that allows you to collect a shared playlist with your loved ones and add it to your music service.

### Main Technologies:

- ReactJS
- Redux Toolkit
- React-Router
- Typescript
- SCSS
- json-server
- Vite

### Local start

1. Install dependencies:

    ```bash
    npm i
    ```
2. Start app: 
    ```bash
    npm start
    ```
